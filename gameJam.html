Game Jam

Descrição

Atenção: este jogo é apenas um aviso!

Estou muito animado para anunciar que este jogo será o resultado da participação na game jam organizada pelo Rickards Game Development Club.

Uma “game jam” é um evento intenso e criativo em que desenvolvedores se reúnem para criar um jogo dentro de um período curto — aprender, experimentar, inovar.

Este site é o meu hub oficial para meus projetos de jogos, e será aqui que publicarei todos os resultados, informações, atualizações e bastidores da participação nessa game jam: desde a concepção até o envio do jogo, além de qualquer detalhe que surgir.

Fique ligado! Mesmo que o jogo não esteja completo agora, esta página serve como o ponto central para acompanhar o que está por vir.
